Markdown Syntax Cheat Sheet

Headings

Use # for headings. The number of # symbols determines the level of the heading.

# H1
    ## H2
    ### H3
    #### H4
    ##### H5
    ###### H6

Text Formatting

Bold: Use **bold** or __bold__

Italic: Use *italic* or _italic_

Strikethrough: Use ~~strikethrough~~

Inline Code: Use `inline code`

Lists

Ordered List

Use numbers followed by a period.

1. First item
    2. Second item
    3. Third item

Unordered List

Use -, *, or + followed by a space.

- First item
    - Second item
    - Third item

Links

Use [link text](URL) to create a link.

[OpenAI](https://www.openai.com)

Images

Use ![alt text](image URL) to insert an image.

![OpenAI Logo](https://openai.com/favicon.ico)

Blockquotes

Use > to create a blockquote.

> This is a blockquote.

Code Blocks

Use triple backticks ``` before and after the code block.

```
    function helloWorld() {
        console.log("Hello, world!");
    }
    ```

Horizontal Rule

Use three or more -, *, or _ on a line by themselves.

---

Tables

Use | to separate columns and - to create headers.

| Syntax | Description |
    | --- | ----------- |
    | Header | Title |
    | Paragraph | Text |

Escaping Characters

Use a backslash \ before the character you want to escape.

\*escaped asterisk\*
